    Игра ДОЛЖНА БЫТЬ УСТАНОВЛЕНА в папке Steam на диске C:
1. Сохранить в безопасное место файл PakList.json из папки с игрой C:\Program Files (x86)\Steam\steamapps\common\PUBG\TslGame\Content\Paks
2. Закинуть пак и файл PakList.json из архива в папкку C:\Program Files (x86)\Steam\steamapps\common\PUBG\TslGame\Content\Paks
3. Проверить, в папке должно быть 300 файла.
4. Запустите файл "regFG.reg".
5. Запустите FGA64.exe.
6. Придумайте мастер пароль.
7. Нажмите на кнопку Restrict.
8. Нажмите на Browse for file и выбрите ранее закинутый пак , нажмите hidden, No access.
9. Закрыть программу, на вопрос Resume the system protection нажать NO.
   
   
   Запуск игры с паками
   
   До запуска игры:
1. Запустить из папки Kill.bat от имени Администратора.
2. Перед запуском игры убедиться, что в папке с паками нет ничего лишнего (должно быть 300 файла)
3. Запустить игру. 

   После запуска игры:
1. Стартануть матч. В игре когда пойдет таймер, Запустите FGA64.exe для скрытия пака,  введите мастер пароль, Нажмите "Protect".
2. Запустить из папки Kill.bat от имени Администратора.
3. Проверить что в папке Paks стало 299 файла:
4. Наслаждайтесь игрой, занимайте топ 1.
5. После того как закончите играть в PUBG не забудьте снять скрытие с пака для последующих запусков. Для этого запустите FGA64.exe, введите мастер пароль, и нажмите кнопку Pause.
6. Перед любым обновлением игры удалить из пакпи созданный пак и заменить на родной файл PakList.json 
7. Запустить из папки с \43tsdg232gf\Kill.bat от имени Администратора.

   Краши после каждой катки
Если у вас присутствую краши после каждой катки, то для этого нужно:
•	Выполнить пункты  из раздела "До запуска игры". 
•	Прогрузитесь в каждую карту (Erangel, Miramar, Sanhok) пока не появится таймер. 
•	Скройте в лобби пак согласно пункту 1 раздела "После запуска игры".	
•	Выполнить оставшиеся пункты с раздела "После запуска игры".
